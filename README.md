# Tampermonkey 脚本集合

这是一个基于 AI 辅助开发的 Tampermonkey (油猴) 脚本仓库。

## 🚀 项目特性
- **零构建**: 纯 JavaScript 实现，即写即用。
- **自动更新**: 脚本通过 GitHub Raw URL 支持 Tampermonkey 的自动更新功能。
- **AI 驱动**: 遵循 `AGENT.md` 规范，由 AI 辅助生成、修改和维护。

## 📦 脚本列表

| 脚本名称 | 描述 | 安装链接 (Raw) | 目标网站 | 版本 |
| :--- | :--- | :--- | :--- | :--- |
| 淘宝/天猫商品详情数据抓取工具 | 一键获取商品数据并发送至指定的 Webhook 地址，支持按钮自由拖拽位置 | [点击安装](./scripts/tm_grabber.user.js) | 淘宝/天猫详情页 | 1.3 |

## 🛠️ 安装说明
1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 在上方的“脚本列表”中找到需要的脚本。
3. 点击“安装链接”跳转至原始代码页面。
4. Tampermonkey 会自动识别并弹出安装界面，点击“安装”即可。

## 📝 开发规范
本项目的所有开发行为必须严格遵守 [AGENT.md](./AGENT.md) 中的指令集。

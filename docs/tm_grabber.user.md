# 淘宝/天猫商品详情数据抓取工具 使用说明

本脚本专为抓取淘宝与天猫商品详情数据并自动化上传至 Webhook 而设计。

## 🌟 核心功能
- **一键提取**：快速解析页面中的 `__ICE_APP_CONTEXT__` 提取商品核心数据。
- **额外信息录入**：支持在提取时补充款式编码与采购人信息（需开启配置）。
- **自动化上传**：支持将数据 POST 到指定的 Webhook 地址，兼容 AirScript。
- **自由交互**：浮动按钮支持自由拖拽并持久化位置。
- **安全鉴权**：支持 `AirScript-Token` 鉴权头。

## 📥 安装方法
1. 确保已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. [点击此处安装脚本](https://raw.githubusercontent.com/Triclouds/TemperMonkey-Scripts/master/scripts/tm_grabber.user.js)。
3. 在弹出的窗口中点击“安装”。

## ⚙️ 配置说明
安装完成后，您需要进行初始配置：
1. 打开任意淘宝或天猫商品详情页。
2. 点击 Tampermonkey 插件图标，选择菜单：**⚙️ 设置提取参数**。
3. 在弹出的对话框中填写以下内容：
   - **用户名**：您的标识符。
   - **Webhook 地址**：接收数据的 API 地址。
   - **AirScript Token**：接口访问鉴权令牌。
   - **启用额外信息录入**：勾选后，每次提取都会弹出补充信息框。
   - **采购人名单**：输入可选的采购人，支持中文逗号、英文逗号或换行分隔。
4. 点击“保存配置”，页面将自动刷新生效。

## 🚀 使用流程
1. **进入详情页**：访问淘宝或天猫的商品详情页面。
2. **定位按钮**：页面右侧会出现一个橙色的“🔍 提取商品JSON”按钮。
3. **点击提取**：
   - 如果开启了“额外信息录入”，会弹出录入框，填写“款式编码”并选择“采购人”。
   - 点击“确认并提取”后，按钮会变为“⏳ 发送中...”。
   - 脚本将提取商品：店铺名称、图片、ID、链接、标题、销量、颜色分类、款式编码、采购人及详细参数列表。
4. **结果提示**：
   - **成功**：弹出“✅ 提取成功！”提示，数据已新增并复制到剪贴板。
   - **重复**：弹出“⚠️ 商品已存在”提示，防止重复录入。
   - **错误**：弹出具体的错误信息，方便排查。

## 📊 数据传输格式 (Webhook)
脚本发送的 POST 请求 Body 格式如下：
```json
{
    "Context": {
        "argv": {
            "username": "你的用户名",
            "grabTime": "2026/3/6 18:18:00",
            "shopName": "店铺名称",
            "images": ["图片链接1", "图片链接2"],
            "itemId": "商品ID",
            "itemLink": "商品链接",
            "title": "商品标题",
            "salesCount": "销量描述",
            "colors": ["颜色1", "颜色2"],
            "styleCode": "款式编码",
            "purchaser": "采购人姓名",
            "parameters": [
                { "label": "参数名", "value": "参数值" }
            ]
        }
    }
}
```

## 🛠️ 常见问题
- **跨域授权提示？** 首次点击提取时，Tampermonkey 可能会弹出“一个用户脚本试图访问跨源资源”的警告。请点击 **“总是允许此域名”** (Always allow all domains)，否则脚本无法将数据发送至 Webhook。
- **按钮没出现？** 请检查当前页面是否为标准的 `item.htm` 商品详情页。
- **提示鉴权失败？** 请检查 Webhook Token 是否配置正确。
- **按钮挡住了内容？** 您可以按住鼠标左键自由移动按钮位置。
